## 三、标签与评测数据集

### 文件列表

| 文件 | 说明 |
|---|---|
| `dataset_split_labels.csv` | 1,957 家样本公司的分类标签，按 7:1:2 切分 Train/Validation/Test |
| `evaluation_ground_truth.csv` | Validation + Test 集中被问询公司的结构化参考答案（监管关注点 JSON） |
| `README.md` | 使用指南 |
| `标签与评测数据集L_数据字典与说明.md` | 字段数据字典 |

### 核心字段

**dataset_split_labels.csv**

| 字段 | 说明 |
|---|---|
| `secucode` | 证券代码 |
| `is_risky` | 预测标签（1=有被问询风险，0=合规安全） |
| `company_type` | 企业类型（国有/民营等） |
| `split` | Train / Validation / Test |

**evaluation_ground_truth.csv**

| 字段 | 说明 |
|---|---|
| `secucode` | 被问询的企业代码 |
| `publish_date` | 问询函下发日期 |
| `announcement_title` | 监管函件官方标题 |
| `regulatory_focus_points_json` | 结构化监管关注点（用于 ROUGE/语义相似度评测） |

## 四、财务市场特征数据集

### 文件列表

| 文件 | 说明 |
|---|---|
| `wind_features_extracted.csv` | 1,951 家公司 × 2020-2024 年各季报期，共 37,222 条记录 |
| `财务市场特征数据集_数据字典与说明.md` | 字段数据字典与缺失说明 |

### 数据来源

| 数据源 | 覆盖公司数 | 说明 |
|---|---|---|
| Wind 金融数据库 | 1,847 家 | 沪深 A 股，全字段 |
| AKShare（同花顺） | 104 家 | 北交所 + B 股，部分字段缺失 |
| 无可用数据 | 6 家 | 疑似退市或数据缺口 |

### 核心字段

| 字段 | 说明 |
|---|---|
| `company_code` | 公司代码（Wind Code） |
| `industry` | 一级行业分类 |
| `report_period` | 报告期（YYYYMMDD） |
| `market_cap` | 总市值（元） |
| `pe_ratio` | 市盈率 TTM |
| `pb_ratio` | 市净率 LF |
| `total_revenue` | 营业总收入（元） |
| `net_profit` | 净利润（归母，元） |
| `operating_cash_flow` | 经营活动现金流量净额（元） |
| `roe` | 净资产收益率（%） |
| `roa` | 总资产收益率（%） |
| `debt_to_assets_ratio` | 资产负债率（%） |
| `revenue_yoy_growth` | 营收同比增长率（%） |
| `net_profit_yoy_growth` | 净利润同比增长率（%） |
