# 财务与市场特征数据集说明文档

本数据集包含 1,951 家比赛目标公司在 2020-2024 年间的历史季度财务与市场估值特征，覆盖 Wind 数据库及 AKShare（同花顺）双数据源。

## 文件说明
- `wind_features_extracted.csv`: 核心特征数据集（CSV 格式，37,222 条记录，1,951 家公司 × 2020-2024 年各季报期）

## 数据来源

| 数据源 | 覆盖公司数 | 说明 |
|---|---|---|
| Wind 金融数据库 | 1,847 家 | 沪深 A 股主力覆盖，全字段 |
| AKShare（同花顺接口） | 104 家 | 北交所 + B 股补充，`roa`/`operating_cash_flow`/估值三列缺失 |
| 无可用数据 | 6 家 | 5 家疑似退市 A 股 + 1 家北交所（832876.BJ），全字段缺失 |

## 数据字典 (Data Dictionary)

| 字段名称 (Column)       | 数据类型 | 含义描述 |
|-------------------------|----------|--------------------------------------------------------------------------|
| `company_code`          | String   | 公司代码（Wind Code，例如 `000004.SZ`） |
| `industry`              | String   | 所属行业分类名称（一级行业，如 `电子`、`航天航空与国防`） |
| `report_period`         | Integer  | 报告期（格式 `YYYYMMDD`，例如 `20231231` 代表 2023 年年报） |
| `market_cap`            | Float    | 报告期季末最后交易日的总市值（Market Capitalization，元） |
| `pe_ratio`              | Float    | 市盈率 TTM（Price-to-Earnings Ratio，滚动市盈率） |
| `pb_ratio`              | Float    | 市净率 LF（Price-to-Book Ratio，最新财报市净率） |
| `total_revenue`         | Float    | 营业总收入（报告期内，元） |
| `net_profit`            | Float    | 净利润（归母净利润，元） |
| `operating_cash_flow`   | Float    | 经营活动产生的现金流量净额（元） |
| `roe`                   | Float    | 净资产收益率，%（Return on Equity） |
| `roa`                   | Float    | 总资产收益率，%（Return on Assets） |
| `debt_to_assets_ratio`  | Float    | 资产负债率，%（总负债 / 总资产） |
| `revenue_yoy_growth`    | Float    | 营业总收入同比增长率，% |
| `net_profit_yoy_growth` | Float    | 净利润同比增长率，% |

## 数据质量与缺失说明

| 字段 | 缺失率 | 原因 |
|---|---|---|
| `market_cap` / `pe_ratio` / `pb_ratio` | 14.7% | Wind 估值表无数据（停牌/新上市/退市）；北交所及 B 股无历史估值截面 |
| `operating_cash_flow` | 5.2% | AKShare 仅提供每股数据，无法换算绝对值 |
| `roa` | 5.2% | AKShare（同花顺）接口未提供该指标 |
| `roe` / `debt_to_assets_ratio` | 2-3% | 新上市公司首期数据缺失 |
| `revenue_yoy_growth` / `net_profit_yoy_growth` | 2-3% | 新上市公司无同比历史基数 |
| `total_revenue` / `net_profit` | <0.2% | 极少量特殊情况 |

**使用建议**：
1. **频率对齐**：估值三列（market_cap/pe_ratio/pb_ratio）取报告期**季末月份最后一个交易日**截面，非固定日历日期，避免了周末/节假日无数据问题。
2. **报表去重**：Wind 多版本报表已去重，每家公司每报告期保留合并报表最新修订版（`ann_dt DESC`）。
3. **缺失填充**：建议对估值三列采用前向填充（ffill）或行业均值填补；roa 可由 `net_profit / (total_revenue / debt_to_assets_ratio * 100)` 近似估算（需配合总资产数据）。
